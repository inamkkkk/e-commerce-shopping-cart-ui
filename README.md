# E-commerce Shopping Cart UI

A Flutter project showcasing a shopping cart UI with quantity controls and total calculation.

## Features

*   Display a list of products.
*   Quantity counter for each product.
*   Real-time total calculation.
*   Responsive and clean UI.

## State Management

The project uses the Provider package for state management.

## Folder Structure


lib/
├── main.dart
├── models/
│   └── product.dart
├── screens/
│   └── cart_screen.dart
├── services/
│   └── cart_service.dart


## Getting Started

1.  Clone the repository.
2.  Run `flutter pub get` to install dependencies.
3.  Run `flutter run` to start the application.
